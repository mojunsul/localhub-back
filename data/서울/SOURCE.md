## 2. 서울시 자치구별 모범음식점 지정 현황

| 항목 | 내용 |
|---|---|
| 파일 수 | 25개 자치구 |
| 지정 기록 수 | 2,964건 |
| 구조 | `DESCRIPTION` + `DATA` |
| 챗봇 카테고리 | 모범음식점 |

업로드된 JSON 자체에는 원본 다운로드 페이지 URL, 라이선스, 취득일 메타데이터가 포함되어 있지 않습니다.
최종 제출 전에 실제 데이터를 취득한 페이지를 기준으로 다음 항목을 기록해야 합니다.

- 제공 기관
- 원본 데이터명
- 원본 URL
- 이용 라이선스
- 데이터 취득일
- 서비스 내 사용 범위

출처와 라이선스를 확인하기 전에는 임의의 URL이나 라이선스를 챗봇 응답에 표시하지 않습니다.
